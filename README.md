# Amazing Task App!
## Features
- It works!
- Lightweight enough for me not to care!
- Only caused major headaches when figuring out why the database wouldn't connect!

This was made in Q1 2026 as a final project for my Intro to Scripting class, where we were tasked with making something using the knowledge given to us throughout the semester. It took me a while to get an initial idea going and by the time I realised the scope of it, the deadline was quickly approaching. In the end I pivoted from making an Escape Room Tracker to a simple To-Do web app. I'm actually quite proud of it, but many sacrifices had to be made to it in the name of getting it done in the first place. I want to revisit this project at some point and remake it in C# but I will tackle that some time after I've taken a break.
